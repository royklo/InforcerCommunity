# Inforcer .NET Builder (deprecated)

The **.NET binary module** (`dotnet/`) has been **removed**. The Inforcer PowerShell module is implemented only as a **script module** under `module/`.

For any changes to Public cmdlets, parameters, output properties, or JSON behavior, use:

- **Unified guardian agent:** `.cursor/agents/inforcer-unified-guardian.md`
- **Guardian skill:** `.cursor/skills/inforcer-unified-guardian/SKILL.md`
- **Findings log:** `FINDINGS.md`

See **CONTRIBUTING.md** and **README.md** for how to build, test, and contribute to the script module.
