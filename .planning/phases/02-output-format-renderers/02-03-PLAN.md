  <action>
    Create `module/Private/ConvertTo-InforcerHtml.ps1` with private function `ConvertTo-InforcerHtml` (per D-01 through D-18, HTML-01 through HTML-10):

    **Function signature:**
    - `[CmdletBinding()]` with comment-based help
    - Single parameter: `[Parameter(Mandatory)][hashtable]$DocModel`
    - Returns: string (complete HTML document)

    **Internal structure (all inside the one .ps1 file):**

    1. **CSS block** (~150-200 lines) as a single-quoted here-string `@' ... '@` stored in `$cssBlock` variable:
       - CSS custom properties on `:root` for light mode: `--bg: #ffffff`, `--text: #1e293b`, `--border: #e2e8f0`, `--row-alt: #f8fafc`, `--header-bg: #f1f5f9`, `--muted: #94a3b8`, `--accent: #3b82f6`, `--badge-bg: #e0f2fe`, `--badge-text: #0369a1`
       - `@media (prefers-color-scheme: dark)` override: `--bg: #0f172a`, `--text: #e2e8f0`, `--border: #334155`, `--row-alt: #1e293b`, `--header-bg: #1e293b`, `--muted: #64748b`, `--accent: #60a5fa`, `--badge-bg: #1e3a5f`, `--badge-text: #93c5fd`
       - `body { font-family: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif; background: var(--bg); color: var(--text); max-width: 1200px; margin: 0 auto; padding: 2rem; line-height: 1.6; }`
       - `table { width: 100%; border-collapse: collapse; margin-bottom: 1rem; }`
       - `th { background: var(--header-bg); text-align: left; padding: 0.5rem 0.75rem; border-bottom: 2px solid var(--border); }`
       - `td { padding: 0.5rem 0.75rem; border-bottom: 1px solid var(--border); }`
       - `tr:nth-child(even) td { background: var(--row-alt); }` (per D-01 alternating rows)
       - `details { margin-bottom: 0.5rem; }`
       - `summary { cursor: pointer; padding: 0.5rem; border-radius: 4px; }` + `summary:hover { background: var(--row-alt); }`
       - `.badge { display: inline-block; background: var(--badge-bg); color: var(--badge-text); padding: 0.125rem 0.5rem; border-radius: 999px; font-size: 0.75rem; margin-left: 0.5rem; }`
       - `.muted { color: var(--muted); font-style: italic; }` (for null/empty em dash values per D-09)
       - `.toc-section { margin-bottom: 1rem; }` `.toc-section a { color: var(--accent); text-decoration: none; }` `.toc-section a:hover { text-decoration: underline; }`
       - `.header { margin-bottom: 2rem; }` `.footer { margin-top: 2rem; padding-top: 1rem; border-top: 1px solid var(--border); color: var(--muted); font-size: 0.875rem; }`
       - `.policy-section { margin-left: 1rem; margin-bottom: 0.5rem; }`
       - `.section-label { font-weight: 600; margin: 0.75rem 0 0.25rem; font-size: 0.875rem; color: var(--muted); text-transform: uppercase; letter-spacing: 0.05em; }`

    2. **StringBuilder assembly** using `$sb = [System.Text.StringBuilder]::new()`:

       a. **HTML head**: `<!DOCTYPE html>`, `<html lang="en">`, `<head>` with `<meta charset="UTF-8">`, `<meta name="viewport" content="width=device-width, initial-scale=1.0">`, `<title>Tenant Documentation - $($DocModel.TenantName)</title>`, `<style>$cssBlock</style>`, `</head>`

       b. **Header section**: `<body>`, `<div class="header">`, `<h1>Tenant Documentation: $($DocModel.TenantName)</h1>`, `<p>Generated: $($DocModel.GeneratedAt.ToString('yyyy-MM-dd HH:mm:ss')) UTC</p>`, if BaselineName: `<p>Baseline: $($DocModel.BaselineName)</p>`, `</div>`

       c. **TOC section** (per D-04, D-05, HTML-02): `<nav class="toc-section">`, `<h2>Table of Contents</h2>`, for each product in `$DocModel.Products.Keys`: `<details>` (NO `open` attribute), `<summary>$prodName</summary>`, `<ul>`, for each category: `<li><a href="#$anchorId">$catName</a></li>`, `</ul>`, `</details>`, end `</nav>`

       d. **Content sections**: for each product: `<h2 id="$prodAnchor">$prodName</h2>`, for each category: `<h3 id="$catAnchor">$catName</h3>`, for each policy: `<details class="policy-section">` (NO `open` attribute per D-06), `<summary>$($policy.Basics.Name) <span class="badge">$($policy.Settings.Count) settings</span></summary>` (per HTML-09)

       e. **Per-policy content** (per HTML-03):
          - **Basics**: `<div class="section-label">Basics</div>`, `<table>`, `<tr><th>Property</th><th>Value</th></tr>`, rows for each Basics key (Description, ProfileType, Platform, Created, Modified, ScopeTags -- skip Name since it is in the summary), null/empty values: `<span class="muted">&mdash;</span>` (per D-09)
          - **Settings** (if count > 0): `<div class="section-label">Settings</div>`, `<table>`, `<tr><th>Setting</th><th>Value</th></tr>`, for each setting: `<tr><td style="padding-left: $($setting.Indent * 1.5)rem">$($setting.Name)</td><td>$value</td></tr>` (per D-07, HTML-04), null/empty Value: `<span class="muted">&mdash;</span>`
          - **Assignments** (if count > 0): `<div class="section-label">Assignments</div>`, `<table>`, `<tr><th>Group</th><th>Filter</th><th>Filter Mode</th><th>Type</th></tr>`, rows from assignment properties, null/empty: `<span class="muted">&mdash;</span>`
          - Close `</details>`

       f. **Footer** (per HTML-07): `<div class="footer">`, `<p>Generated: $($DocModel.GeneratedAt.ToString('yyyy-MM-dd HH:mm:ss')) UTC | Tenant: $($DocModel.TenantName)</p>`, `</div>`, `</body>`, `</html>`

    3. **Return**: `$sb.ToString()`

    **Anchor ID generation**: Use a helper function `ConvertTo-HtmlAnchorId` inside the file that converts text to lowercase, replaces spaces/special chars with hyphens, strips non-alphanumeric except hyphens. For uniqueness across products + categories, prefix category anchors with product name: `"$prodName-$catName"` then convert.

    **HTML entity escaping**: Escape `<`, `>`, `&`, `"` in all user-provided values (policy names, setting names/values) using `[System.Net.WebUtility]::HtmlEncode()` before appending to StringBuilder.

    **All `$sb.Append*` calls MUST use `[void]` prefix** to suppress pipeline output.

    **No file I/O, no API calls, no JavaScript.**
