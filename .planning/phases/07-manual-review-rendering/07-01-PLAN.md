__DUPLICATE_TABLE__ data contract (from Phase 4):
- Value starts with `__DUPLICATE_TABLE__` prefix
- After stripping prefix, JSON array of @{Policy; Value; Side} objects
- Example: `__DUPLICATE_TABLE__[{"Policy":"Policy A","Value":"true","Side":"Source"},{"Policy":"Policy B","Value":"false","Side":"Destination"}]`

Compliance rules JSON shape (from 07-RESEARCH.md):
- Root has `Rules` or `rules` key containing an array
- Each rule: settingName/SettingName, operator/Operator, dataType/DataType, operand/Operand
- Example: `{"Rules":[{"settingName":"BitLocker","operator":"IsEquals","dataType":"Boolean","operand":"true"}]}`
